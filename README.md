# WB SEO Generator

AI-инструмент для создания продающих карточек Wildberries.

---

## Быстрый старт

### Windows
Дважды кликните на файл **ЗАПУСТИТЬ.bat**

### macOS / Linux
```bash
chmod +x запустить.sh
./запустить.sh
```

---

## Требования

**Node.js** — нужен для запуска прокси-сервера MPStats.

Скачать: https://nodejs.org (выберите версию LTS)

> Без Node.js программа всё равно работает — через публичный API Wildberries.
> Node.js нужен только для полных данных MPStats (продажи, выручка).

## API ключи (вводятся в настройках программы)

| Сервис | Для чего | Где получить |
|--------|----------|--------------|
| **MPStats** | Данные конкурентов (продажи, выручка) | mpstats.io → Личный кабинет → API |
| **Perplexity** | Генерация SEO-описания | perplexity.ai → Settings → API |
| **OpenAI** | Генерация изображений (GPT Image 1) | platform.openai.com → API keys |
| **OpenRouter** | Генерация изображений (Nano Banana) | openrouter.ai → Keys |

## Как пользоваться

1. Запустите программу через **ЗАПУСТИТЬ.bat** (Windows) или **запустить.sh** (Mac/Linux)
2. Нажмите кнопку **Настройки** → введите API ключи
3. Если есть MPStats — включите **CORS-прокси** в настройках
4. Введите название товара, загрузите фото
5. Нажмите **Найти категории** → выберите категорию
6. Нажмите **Анализ конкурентов**
7. Нажмите **Генерировать SEO + изображения**
8. На вкладке **Изображения** выберите концепции съёмки → генерируйте

---

## Если MPStats не работает

MPStats блокирует прямые запросы из браузера (CORS).
Прокси-сервер (`proxy.js`) решает эту проблему — он запускается локально
и перенаправляет запросы к MPStats.

**Без прокси** программа автоматически использует публичный API Wildberries —
это бесплатно и не требует ключей, но без данных о продажах.
