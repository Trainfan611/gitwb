# WB SEO Generator

AI-инструмент для создания продающих карточек Wildberries.

---

## Быстрый старт

```bash
npm install
npm start
```

После запуска откройте `http://localhost:3000`.

---

## Требования

**Node.js** — нужен для запуска приложения и встроенного MPStats-прокси.

Скачать: https://nodejs.org (выберите версию LTS)

> Без MPStats-ключа программа всё равно работает через публичный API Wildberries.

## Автоподстановка ключей при старте

Чтобы ключи были заполнены сразу при открытии приложения, добавьте в Railway Variables:

- `APP_PREFILL_KEYS=true`
- `MPSTATS_API_KEY=...`
- `PERPLEXITY_API_KEY=...`
- `OPENAI_API_KEY=...`
- `OPENROUTER_API_KEY=...` (опционально)
- `AUTO_ENABLE_MPSTATS_PROXY=true` (опционально, чтобы MPStats-прокси включался автоматически)

## Основные файлы

- `index.html` — интерфейс WB SEO Generator
- `server.js` — статический сервер + MPStats-прокси (`/api/mpstats/*`)

---

## API ключи (вводятся в настройках программы)

| Сервис | Для чего | Где получить |
|--------|----------|--------------|
| **MPStats** | Данные конкурентов (продажи, выручка) | mpstats.io → Личный кабинет → API |
| **Perplexity** | Генерация SEO-описания (ключ `pplx-...`) | perplexity.ai → Settings → API |
| **OpenAI** | SEO через ChatGPT (`gpt-4o-mini`) и изображения (GPT Image 1) | platform.openai.com → API keys |

SEO: сначала вызывается Perplexity (если указан ключ `pplx-...`), при ошибке или отсутствии ключа используется OpenAI. Ключ `sk-...`, вставленный в поле Perplexity, автоматически считается ключом OpenAI.
| **OpenRouter** | Генерация изображений (Nano Banana) | openrouter.ai → Keys |

## Как пользоваться

1. Запустите программу через **ЗАПУСТИТЬ.bat** (Windows) или **запустить.sh** (Mac/Linux)
2. Нажмите кнопку **Настройки** → введите API ключи
3. Если есть MPStats — включите прокси в настройках (используется встроенный серверный прокси)
4. Введите название товара, загрузите фото
5. Нажмите **Найти категории** → выберите категорию
6. Нажмите **Анализ конкурентов**
7. Нажмите **Генерировать SEO + изображения**
8. На вкладке **Изображения** выберите концепции съёмки → генерируйте

---

## Если MPStats не работает

MPStats блокирует прямые запросы из браузера (CORS).
В проекте уже есть встроенный прокси в `server.js`, отдельный запуск `proxy.js` не нужен.

**Без прокси** программа автоматически использует публичный API Wildberries —
это бесплатно и не требует ключей, но без данных о продажах.
